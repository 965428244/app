Remote View sessions

Remote View is the easiest and fastest way to preview and live edit your local web-site on any internet-connected browser or device.

Remote View creates a publicly available domain name like http://some-name.livestyle.io and connects it with your local web-site (for example, http://localhost:8080). So it looks like anyone browsing the public domain is actually browsing your local web-site from your computer.

Share this public URL with colleagues or customers, preview it on any internet-connected mobile device, virtual machine and so on—with instant LiveStyle updates.

To start a Remote View session, go to your local web-site in Google Chrome browser, open LiveStyle popup and toggle ”Remote View” button.

Remote View is a paid service available for free during beta-testing.

Sublime Text plugin installation

Install via Package Control (recommended)

You must have Package Control installed first.

  1. In Sublime Text, open Command Palette with Tools > Command Palette... menu item.
  2. Type add repository to find Package Control: Add Repository command and hit Enter.
  3. Paste the following URL into the opened panel: https://github.com/livestyle/sublime-text. Hit Enter.

Wait a few moments until extension is installed.

Manual installation

  1. In Sublime Text, pick Preference > Browse Packages... menu item to open Packages folder.
  2. Create LiveStyle folder in it.
  3. Download package ZIP and extract its contents into a newly created LiveStyle folder. Note that in most cases the downloaded ZIP will be unpacked into a sub-folder: you have to copy files from that sub-folder into LiveStyle folder you’ve created before.
  4. Restart Sublime Text.
